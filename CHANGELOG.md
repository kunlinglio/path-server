# Changelog

All notable changes to the Path Server will be documented in this file.

## [0.2.0] - 2026-03-04
### Added
- Initial release of VS Code extension with language-server support.

### Changed
- Refactored repository layout to a monorepo (consolidated `path-server-zed` and `path-server-vscode` into `path-server`).
- Refactored completion logic for improved accuracy and maintainability.
- Improved README readability.

## [0.1.0] - 2026-03-03
Initial release of Path Server.

### Added
- Support path completion, both relative and absolute paths.
- Support relative path based on workspace root or current document.
